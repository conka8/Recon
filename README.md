## testing and deployment
#### clone project and install dependencies

```bash
npm install
```
#### to run tests:

```bash
npx hardhat test
```
#### update .env file 


#### deployment:
Just run hardhat run with desired Network

```bash
npx hardhat run --network <your-network> scripts/deployReconNetwork.js
```
